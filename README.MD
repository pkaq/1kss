## Usage

####注意:只支持cent7以上发行版


**Via curl**
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/pkaq/1kss/master/1kss.sh)"
```

**Via wget**   


```

sh -c "$(wget https://raw.githubusercontent.com/pkaq/1kss/master/1kss.sh -O -)"

```
